STEP1　基礎知識の習得

自己学習のためのテキスト・動画を提供。

分からなかったところは
オンラインのマンツーマンレッスンで
理解できるまで教えます。

STEP2　開発体験

実際の現場を想定し、クライアントの要望を要件書に落とし込み、実際にアプリとして開発体験をしていきます。

この段階で実際に案件を受注できるレベルに到達。

STEP3　案件獲得

実際に案件を受注し、こなしていきましょう。
案件の獲得方法、アピールの仕方もお教えします。

分からないところはしっかりフォローします。

現役のフリーランスエンジニアで、大手プログラミングスクールで講師も務めています。

このスクールではカリキュラムが終了した際に、フリーランスとして独り立ちできることを目標に掲げています。
会社員を抜け出して、それぞれが思い描いた生活をしてほしいと思っているからです。

そして、少しだけ個人的なことを伝えさせていただくと、このスクールを通じて優秀なエンジニアが一人でも増えてほしいと思っています。
ボクもまだまだたくさんやりたいことがあります。
そのためにたくさんのエンジニア仲間が必要だと思っています。
このスクールを通じて、一緒に働く仲間が見つかればすごく嬉しいです。
今は一人で仕事をしていますが、優秀な人達とチームを組んで人の役に立てるような大きな仕事をすることが夢なんです。

このスクールを受講してくださった方、必ず一人前のエンジニアに育て上げます。
みなさんの夢を叶えるためにも、ぜひボクと一緒に頑張りましょう！
